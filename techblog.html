PIC Microcontroller

PIC - Harvard architecture microcontroller, Wikipedia says "generally thought to mean Peripheral Interface Controller, was initially Programmable Interface Controller, and currently stands for Programmable Intelligent Computer"

Interfacing a Flash Memory

Flash memory can be interfaced using SPI protocol. The flash behaves like a slave, while the PIC is a master. The 4 lines required are Serial Input (SI), Serial Output (SO), Clock (SCK) and Chip Select or Slave Select (SS). PPS register in the PIC gives the abitlity to configure the pins to the SI, SO, SCK. The SS can be any General Purspose input-output pin (GPIO).
