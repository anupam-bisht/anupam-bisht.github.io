Update 29th Dec 2020: Wifi connection to rpi

  • MATLAB advices to configure rpi settings through command line. I noticed a error "Illegal instruction" while doing that. However, doing mypi = raspi(ipaddress,username,password) worked in MATLAB. ipaddress can be found by usig ifconfig in shell for rpi (via puTTY or related).My ip looks like 192.X....
  • To enable Serial peripheral interface(SPI) in MATLAB you need to go to the shell again and run sudo raspi-config, go to interfacing options and find SPI, follow the steps. This can be checked in MATLAB mypi.AvailableSPIChannels .
