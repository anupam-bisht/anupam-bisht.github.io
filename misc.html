Photoplethysmograph (PPG)

At Vellore institute of technolgy (VIT), we had some courses which had project courses. This was one of the project. It has been uploaded as a video in youtube. Click here for video

The project used LIFA - Labview Interface for Ardunio. The main idea behind the Labview VI was to place a "inti", "analog read", and a "close block" in a loop (similar to any sensor reading tutorial in youtube, e.g. Arduino meets LabVIEW: Reading Analog Signals of POT in LabVIEW). However, a better way to do this will be to use a protocol such as SPI or I2C which will allow time locked data aquisition.
